# Mini SEIM Dashboard

A web-based Security Information and Event Management (SEIM) Dashboard built using Flask and MySQL.

## Features

- User Authentication
- Log Upload
- Log Parsing
- Threat Detection
- Dashboard Analytics
- Alert Managment
- PDF & CSV Reports

## Technology Stack

- Python
- Flask
- MySQL
- Bootsrap 5
- Chart.js
- SQLAlchemy
- Flask-Login
- Pandas
- ReportLab
